# BiteRush - Proiect Tehnici Web (Frontend)

BiteRush este un website de prezentare responsiv creat pentru un restaurant, oferind o interfață modernă și prietenoasă pentru utilizatori. Proiectul include pagini pentru meniu, detalii despre servicii, date de contact și un sistem dinamic de autentificare/înregistrare.

## 🍔 Caracteristici

- **Design Responsiv**: Construit folosind metodologia "Mobile First", adaptabil și compatibil cu orice dispozitiv mobil sau desktop.
- **Navigare Intuitivă**: Bară de navigare cu meniu de tip hamburger pentru ecrane mici și meniu dropdown pentru categoriile produselor (ex. Chicken burger, Hamburger, Aripioare de pui).
- **Secțiuni Integrate**: 
  - *Acasă* - Prezentarea principală "Mâncarea este pe primul loc".
  - *Despre Noi* - Povestea restaurantului BiteRush.
  - *Servicii* - Informații despre comenzi, livrări și servicii la ușă.
- **Autentificare**: Formular interactiv de Login & Înregistrare, care permite comutarea fluidă între conectare și crearea unui cont nou. Conține etichete flotante (floating labels) și un design atrăgător.

## 🛠️ Tehnologii Utilizate

- **HTML5** - Pentru structurarea semantică a paginilor.
- **CSS3** - Pentru stilizare, animații și aspectul responsiv (media queries).
- **JavaScript** - Pentru funcționalitatea meniului responsiv și interactivitatea formularului de conectare/înregistrare.
- **FontAwesome** - Integrat pentru iconițele vectoriale din cadrul site-ului (social media, butoane meniu).
